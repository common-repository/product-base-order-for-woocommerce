Product Base Order for WooCommerce

== Description ==

Product Base Order for WooCommerce, using this plugin visitors or logged users can see the order listing by shortcodes and single products pages. Can easily manage table view columns dynamically also able to filtering by products and category wise. Manage columns title by using shortcodes.

= Basic Features =

* Able to see orders listing by products and category wise
* Manage table view columns from admin settings
* Have option only can see logged users only
* Key word searching, shorting, pagination manage on orders table view
* Easy use by shortcodes
* Easy admin settings for manage labels and others options

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
